# 🎯 GUI增强版 - 表格对比功能

## ✅ **新增功能**

### 📊 **详细对比表格**
- **第1列**: 输入数据（截断显示，悬停查看完整内容）
- **第2列**: 版本A输出结果
- **第3列**: 版本B输出结果  
- **第4列**: 差异百分比（带颜色编码）

### 🎨 **颜色编码**
- 🟢 **绿色**: 0% 差异（完全相同）
- 🟡 **黄色**: < 20% 差异（轻微差异）
- 🔴 **红色**: ≥ 20% 差异（重大差异）

### 📱 **界面布局**
- **左侧** (350px): 简单测试状态文本
- **右侧** (650px): 详细对比表格
- **可调整分割器**: 拖动调整左右比例

## 🚀 **使用方法**

### 1. 启动GUI
```bash
python main.py
```

### 2. 配置测试
- **版本A**: 浏览选择第一个C文件
- **版本B**: 浏览选择第二个C文件
- **测试数据**: 选择包含测试文件的目录

### 3. 查看结果
- **左侧文本区**: 显示每个测试的通过/失败状态
- **右侧表格**: 显示详细的输入输出对比

### 4. 分析差异
- **悬停查看**: 鼠标悬停在表格单元格上查看完整内容
- **颜色判断**: 通过颜色快速判断差异程度
- **百分比量化**: 精确的差异百分比计算

## 🔧 **差异计算算法**

### 字符级差异分析
```python
# 计算字符差异百分比
max_len = max(len(output_a), len(output_b))
if max_len > 0:
    diff_chars = sum(1 for i in range(min_len) if output_a[i] != output_b[i])
    diff_chars += abs(len(output_a) - len(output_b))
    diff_percent = (diff_chars / max_len) * 100
```

### 示例解释
- **0%**: 完全相同的输出
- **15%**: 轻微差异，可能是格式问题
- **50%**: 中等差异，可能有逻辑差异
- **100%**: 完全不同的输出
